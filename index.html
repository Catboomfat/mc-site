Commands

Main Commands

/trigger sethome Used to set home. Note: If you change username, your last sethome position will be lost
/trigger home Used to teleport home instantly.
/trigger spawn Used to teleport to spawn instantly.
/trigger wild Used to teleport to a random wild location. Note: Do not move after typing command
/trigger tpa set # Used to teleport to in-game player. Note: # is the player number, which can be found on the right of the players username by holding tab
/trigger setreturn Set XYZ coordinates for the command /trigger return. Note: Coordinates will be overridden if you teleport to home or spawn
/trigger return Return to XYZ coordinates See: /trigger setreturn
/trigger warp set #ID Warps you to #ID #ID is from 1-8
